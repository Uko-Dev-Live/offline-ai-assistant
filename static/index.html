Ready when you are.

This assistant runs entirely on your machine. No data leaves this computer.

Structured output

The model is forced to return JSON matching a Pydantic schema (answer, confidence, reasoning, follow_up_questions, tags). On invalid output the server retries once before giving up gracefully.

Benchmark

Quick measurement of time to first token (TTFT), tokens / second and total latency. For repeatable results use the CLI: python -m scripts.benchmark.
